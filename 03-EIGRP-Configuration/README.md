This project demonstrates the configuration of an advanced dynamic routing environment using Cisco's EIGRP (Enhanced Interior Gateway Routing Protocol). By assigning all four routers to the same Autonomous System, the network uses the DUAL (Diffusing Update Algorithm) to rapidly discover loop-free paths and calculate composite metrics based on bandwidth and delay. This setup ensures incredibly fast network convergence, automatically maintains backup routes (feasible successors), and dynamically scales traffic across the topology far more efficiently than standard distance-vector protocols.
