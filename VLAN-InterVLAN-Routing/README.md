This project demonstrates local area network segmentation and Inter-VLAN routing across multiple switches. By separating the hosts into two distinct logical networks—VLAN 10 (10.0.0.0 subnet) and VLAN 20 (20.0.0.0 subnet)—broadcast domains are isolated to significantly improve network security and efficiency. An 802.1Q trunk link allows traffic from both VLANs to traverse the connection between the switches, while a central router configured with sub-interfaces (Router-on-a-Stick) handles the Layer 3 routing to enable successful communication between the two separate network segments.
